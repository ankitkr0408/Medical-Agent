## 🚀 Deployment Steps (Quick Reference)

### 1. Run Pre-Deployment Check
```bash
cd medical-platform-nextjs
npm run deploy:check
```

### 2. Push to GitHub
```bash
git add .
git commit -m "Ready for Vercel deployment"
git push origin main
```

### 3. Deploy to Vercel
1. Go to [vercel.com/new](https://vercel.com/new)
2. Import your GitHub repository
3. **Set Root Directory:** `medical-platform-nextjs`
4. Add 3 environment variables
5. Click Deploy

### 4. Post-Deployment
1. Add `NEXTAUTH_URL` with your Vercel URL
2. Configure MongoDB Atlas (allow 0.0.0.0/0)
3. Redeploy
4. Test everything

## 🆘 Quick Troubleshooting

### Build Fails
```bash
# Ensure dependencies are installed
npm install
npm run build
```

### Can't Login After Deployment
```bash
# Add NEXTAUTH_URL in Vercel dashboard
NEXTAUTH_URL=https://your-vercel-url.vercel.app
# Then redeploy
```

### Database Connection Error
1. Check MongoDB Atlas Network Access
2. Allow 0.0.0.0/0 (all IPs)
3. Verify DATABASE_URL is correct

### AI Analysis Not Working
1. Check OpenAI API key is valid
2. Verify you have credits
3. Check Vercel function logs
